## Table of contents

## Capabilities
* **Lock** Home Arm
* **Switch** Full Arm(away) / DisArm
* **Disarm**
* **Refresh**
* **Sstatus**

## Troubleshooting

* **Home arm** - use 'lock door'
* **Full arm** - use 'Switch On'
* **Disarm** - use 'Switch OFF'
